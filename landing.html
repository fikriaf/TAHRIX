Arsitektur Sistem

Enam Layer Terisolasi, Satu Tujuan

Setiap layer memiliki tanggung jawab yang terisolasi dan berkomunikasi melalui interface yang terdefinisi.

L1
Storage
Neo4j AuraDB untuk graph traversal, PostgreSQL untuk case management, Redis untuk caching, dan IPFS untuk evidence immutable.
Neo4j PostgreSQL Redis IPFS
L2
Ingestion
Multi-chain indexer dengan adapter per blockchain. Real-time stream processor via Helius Webhooks dan Alchemy WebSocket.
Alchemy Helius WebSocket
L3
Integration
API Gateway dengan rate limiting, request validation, structured logging, dan circuit breaker. Cross-chain bridge adapter.
FastAPI Celery asyncio
L4
AI Core
Agent Orchestrator dengan ReAct loop, Trace Engine untuk graph traversal, GNN Engine (GAT 3-layer), dan Anomaly Detector 17 kategori.
OpenAI PyG ONNX
L5
Intelligence
Risk Scorer agregasi multi-sinyal, XAI module dengan SHAP values, dan Entity Resolution Engine untuk link on-chain ke off-chain.
SHAP XAI
L6
Output
Risk Dashboard interaktif, Compliance Report Generator PDF, dan Evidence pinning ke IPFS dengan CID yang tersimpan di PostgreSQL.
D3.js WeasyPrint

Cognitive Loop

Think → Act → Observe → Reflect

Paradigma ReAct yang memungkinkan AI merumuskan hipotesis, mengeksekusi investigasi, dan merefleksi hasil secara mandiri.

THINK
Rumuskan hipotesis investigasi berdasarkan input wallet atau TX hash
→
ACT
Eksekusi tool: sanctions check, transaction fetch, GNN inference, graph trace
→
OBSERVE
Kumpulkan observasi: anomaly flags, entity labels, risk indicators
→
REFLECT
Evaluasi stopping criterion: max 5 iterasi, confidence > 0.85, critical finding
Stopping criteria terpenuhi? GENERATE REPORT · Belum? LANJUT THINK
